# UrbanHarmony Personal Website

This project satisfies the personal website final requirements:

- Static website ready for GitHub Pages
- Professional, responsive design
- Google Analytics script included in `index.html`
- Booking modal includes Google Analytics events for interaction tracking
- Website can be demoed live after GitHub Pages deployment

## Important Google Analytics Step

In `index.html`, replace both appearances of this placeholder:

```text
G-XXXXXXXXXX
```

with your real Google Analytics Measurement ID.

Your Measurement ID usually looks like:

```text
G-ABC123XYZ0
```

## Deploy on GitHub Pages

1. Push the `urbanharmony_refined` files to your GitHub repository.
2. Go to the repository on GitHub.
3. Open **Settings**.
4. Click **Pages**.
5. Under **Build and deployment**, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
6. Save and wait for GitHub to publish the website.
7. Open the published GitHub Pages link and test the site.

## Demo Checklist

- Open the live GitHub Pages website.
- Show the homepage, navigation, sections, images, and booking modal.
- Show that Google Analytics code exists in `index.html`.
- Open Google Analytics and show website traffic/statistics.
- If there is no traffic yet, open the live website in a browser/incognito window and refresh it, then check the Realtime report.
